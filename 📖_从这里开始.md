## 💡 关键信息卡片

### 项目类型
🎮 嵌入式系统黑盒测试 - 温度控制器

### 核心技术
```
Lua脚本 + JSON协议定义 + YAML测试数据 + 串口通信
```

### 三大协议
```
P_Sensor (0x01,0x10) → 温度传感器输入
P_Heater (0x02,0x11) → 加热棒输出
P_Fan    (0x02,0x22) → 散热风扇输出
```

### 必记公式
```lua
function entry()
    clear(channels.XXX)
    write_msg(channels.XXX, protocols.P_XXX, {data})
    res = read_buff(channels.XXX, 0, timeout)
    pkg = unpack(protocols.P_XXX, ebuff.from_buff(res.value), true)
    if pkg then check(condition, "pass", "fail") end
    exit()
end
```

### 测试三件套
```
XX_名称.lua  → 测试脚本
XX_名称.yml  → 测试数据（可选）
XX_名称.run  → 执行配置
```

## ⚠️ 重要提示

### ✅ 一定要做的
1. ✅ 所有路径调用 `exit()`
2. ✅ 解包前检查 `if pkg then`
3. ✅ 延时要充足（至少1500ms）
4. ✅ 使用 `on_buff_recv` 后记得 `off_recv`
5. ✅ 大量使用 `print()` 调试

### ❌ 一定不要做的
1. ❌ 不检查就访问 `pkg.value`
2. ❌ 延时太短（<1000ms）
3. ❌ 忘记调用 `exit()`
4. ❌ 回调函数内做耗时操作
5. ❌ 混淆Lua数组索引（从1开始）

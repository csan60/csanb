## 💡 核心知识点

### 项目本质
```
嵌入式系统黑盒测试
使用Lua脚本通过串口测试温度控制器
```

### 技术栈
```
Lua (测试脚本)
+ JSON (协议定义 + 执行配置)
+ YAML (测试数据)
+ 串口RS232 (通信接口)
```

### 三大协议
```
P_Sensor → 温度传感器输入 (0x01,0x10)
P_Heater → 加热棒输出     (0x02,0x11)
P_Fan    → 散热风扇输出   (0x02,0x22)
```

### 协议统一格式
```
[包头0xFFFA大端] [类型1] [类型2] [长度]
[数据字段小端] [校验SUM_8_O16小端] [包尾0x0F]
```

### 核心API（必记5个）
```lua
clear(channels.XXX)              -- 清空
write_msg(ch, prot, data)        -- 发送
read_buff(ch, 0, timeout)        -- 读取
unpack(prot, buffer, true)       -- 解包
exit()                           -- 退出
```

### 测试三件套
```
XX_名称.lua  -- 测试脚本（必须）
XX_名称.yml  -- 测试数据（可选）
XX_名称.run  -- 执行配置（必须）
```

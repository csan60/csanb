# 云端注入器(CxExam为例)

## 简介
一些外挂的运行模式--注入器模式（云端注入），本程序作为一个示例以CxExam为例，展示如何实现注入器功能。

DLL文件来源于[cef_cx_copy_tool](https://github.com/SJYssr/cef_cx_copy_tool)

## 技术解析
dll文件保存在服务器，注入器启动后找目标文件然后将云端的dll下载到目标文件同级文件夹，隐藏注入器，监听目标程序，目标程序关闭清理dll关闭注入器

## 功能

- 支持手动定位或自动查找目标文件文件
- 自动下载必要的运行资源
- ✨**新增**：支持自动隐藏到系统托盘
- ✨**新增**：集成AI答题助手功能
- ✨**新增**：支持OpenAI、DeepSeek等AI接口
- 启动后可选自动隐藏并监听目标进程
- 目标进程结束后自动清理资源并退出

## 使用说明

1. 选择 CXExam.exe 文件的路径：
   - 点击"浏览..."手动选择文件
   - 或点击"自动查找"让程序自动搜索
2. 点击"启动"按钮：
   - 程序会自动下载必要资源
   - 启动考试系统
   - 主程序会自动隐藏
3. 目标程序关闭后，注入器会自动清理资源并退出

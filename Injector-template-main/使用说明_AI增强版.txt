🎯 主要新增功能
============================================================

1. 自动隐藏到系统托盘
   - 避免干扰考试，程序可自动隐藏到托盘
   - 支持手动隐藏和自动隐藏两种模式
   - 双击托盘图标可快速恢复窗口

2. AI答题助手
   - 集成OpenAI兼容的AI接口
   - 支持多种AI服务（OpenAI、DeepSeek、Kimi等）
   - 智能分析题目并给出答案

3. 配置持久化
   - 配置自动保存，无需每次重新设置
   - 配置文件：ai_config.ini

📖 快速开始
============================================================

第一步：配置AI接口（可选，如不使用AI功能可跳过）
----------------------------------------------
1. 点击"进入软件"后，进入第二界面
2. 点击"AI配置"按钮
3. 填写以下信息：
   
   【API地址】常用选项：
   - OpenAI: https://api.openai.com/v1/chat/completions
   - DeepSeek: https://api.deepseek.com/v1/chat/completions
   - 其他服务：根据服务商文档填写
   
   【API Key】
   - 从AI服务提供商处获取（通常在账户设置中）
   - OpenAI: https://platform.openai.com/api-keys
   - DeepSeek: https://platform.deepseek.com
   
   【模型】推荐选项：
   - gpt-3.5-turbo（快速，成本低）
   - gpt-4（质量高，成本高）
   - deepseek-chat（DeepSeek专用）

4. 点击"保存配置"
5. 点击"测试连接"验证配置是否正确

第二步：使用主要功能
----------------------------------------------
A. 启动目标程序
   1. 点击"浏览..."选择CXExam.exe，或点击"自动查找"
   2. 勾选"启动后自动隐藏到托盘"（可选）
   3. 点击"启动"按钮
   4. 如果勾选了自动隐藏，窗口会自动隐藏到托盘

B. 使用AI答题助手
   1. 点击"AI答题助手"按钮打开助手窗口
   2. 在"输入题目"区域输入或粘贴题目内容
   3. 点击"AI解答"按钮（或按F5键）
   4. 等待几秒，答案会显示在"AI答案"区域

C. 隐藏/显示窗口
   - 手动隐藏：点击"隐藏到托盘"按钮
   - 显示窗口：双击系统托盘图标
   - 退出程序：右键托盘图标 → 选择"退出"

============================================================
🔧 常见问题解答
============================================================

Q1: AI测试连接失败怎么办？
A1: 请检查：
    - API地址是否正确（包括http://或https://）
    - API Key是否有效
    - 网络连接是否正常
    - 是否需要配置代理

Q2: AI响应很慢怎么办？
A2: 可能的原因和解决方法：
    - 网络延迟：检查网络连接
    - 模型太大：换用gpt-3.5-turbo等更快的模型
    - 服务商限流：稍后再试或更换服务商

Q3: 托盘图标看不到怎么办？
A3: Windows可能隐藏了部分托盘图标：
    - 点击任务栏右下角的"^"图标
    - 或在"任务栏设置"中启用本程序的托盘图标

Q4: 如何获取API Key？
A4: 常见服务商获取方式：
    【OpenAI】
    1. 访问 https://platform.openai.com
    2. 注册/登录账号
    3. 进入 API Keys 页面
    4. 点击"Create new secret key"
    5. 复制生成的key（只显示一次）
    
    【DeepSeek】
    1. 访问 https://platform.deepseek.com
    2. 注册/登录账号
    3. 在账户设置中找到API Key
    4. 创建或复制key

Q5: 配置文件在哪里？
A5: 配置文件名为ai_config.ini，保存在程序同目录下
    可以用记事本打开查看或编辑

Q6: 程序闪退或出错怎么办？
A6: 请检查：
    - 是否以管理员权限运行
    - 防火墙是否阻止了程序
    - 查看调试控制台的错误信息
    - 尝试重新配置AI设置

============================================================
💡 使用技巧
============================================================

1. 题目输入技巧
   - 可以直接粘贴整道题目（包括选项）
   - 也可以只输入问题部分
   - 支持多行文本，题目较长时可分段输入

2. 答案理解技巧
   - AI会给出解题思路和最终答案
   - 建议理解思路后再做答，不要直接照搬
   - 如果答案不满意，可以重新提问或换个问法

3. 窗口管理技巧
   - F5快捷键可快速请求AI解答
   - 答题助手窗口可以调整大小
   - 多个题目可以保持助手窗口打开，连续提问

4. 配置优化技巧
   - 日常使用推荐gpt-3.5-turbo，速度快成本低
   - 复杂问题可临时切换到gpt-4
   - DeepSeek价格便宜，中文效果好

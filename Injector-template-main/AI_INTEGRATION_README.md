# 智能注入器 - AI集成增强版

## 🚀 新增功能

### 1. 自动隐藏到系统托盘
- **功能说明**：启动目标程序后，主窗口可自动隐藏到系统托盘，避免影响考试
- **使用方式**：
  - 在第二界面勾选"启动后自动隐藏到托盘"复选框
  - 或点击"隐藏到托盘"按钮手动隐藏
  - 双击托盘图标或右键菜单可恢复窗口
- **快捷操作**：
  - 双击托盘图标：显示主窗口
  - 右键托盘图标：弹出菜单（显示窗口/退出）

### 2. AI答题助手
- **功能说明**：集成AI接口，可以帮助理解和解答考试题目
- **支持的AI服务**：
  - OpenAI（GPT-3.5/GPT-4）
  - DeepSeek
  - Kimi
  - 其他OpenAI兼容接口

## 📖 使用教程

### 配置AI接口

1. 点击"AI配置"按钮打开配置窗口
2. 填写以下信息：
   - **API地址**：AI服务的API端点
     - OpenAI: `https://api.openai.com/v1/chat/completions`
     - DeepSeek: `https://api.deepseek.com/v1/chat/completions`
     - 自定义接口：根据提供商文档填写
   - **API Key**：从AI服务提供商获取的密钥
   - **模型**：选择使用的AI模型
     - OpenAI推荐：`gpt-3.5-turbo`（快速）或 `gpt-4`（高质量）
     - DeepSeek推荐：`deepseek-chat`
3. 点击"保存配置"保存设置
4. 点击"测试连接"验证配置是否正确

### 使用AI答题助手

1. 点击"AI答题助手"按钮打开助手窗口
2. 在"输入题目"文本框中输入或粘贴题目内容
3. 点击"AI解答"按钮（或按F5键）
4. 等待AI分析，答案会显示在下方的"AI答案"区域

### 隐藏/显示窗口

**方法一：自动隐藏**
1. 勾选"启动后自动隐藏到托盘"
2. 点击"启动"按钮
3. 程序会自动隐藏到系统托盘

**方法二：手动隐藏**
1. 点击"隐藏到托盘"按钮
2. 主窗口立即隐藏到系统托盘

**恢复窗口**
- 双击系统托盘图标
- 或右键点击托盘图标，选择"显示窗口"

## ⚙️ 配置文件

配置保存在程序同目录下的 `ai_config.ini` 文件中：

```ini
[AI]
Endpoint=https://api.openai.com/v1/chat/completions
ApiKey=sk-your-api-key-here
Model=gpt-3.5-turbo

[Settings]
AutoHide=1
```

## 🛠️ 常见问题

### Q1: AI接口测试失败？
**A**: 检查以下几点：
- API地址是否正确
- API Key是否有效
- 网络连接是否正常
- 是否需要配置代理

### Q2: AI响应很慢？
**A**: 可能原因：
- 网络延迟较高
- 使用的模型较大（如GPT-4）
- API服务商限流
- 建议使用更快的模型（如gpt-3.5-turbo）

### Q3: 托盘图标不显示？
**A**: 可能原因：
- Windows设置隐藏了某些托盘图标
- 打开"任务栏设置" > "选择在任务栏上显示哪些图标"
- 启用本程序的托盘图标显示

### Q4: 如何获取API Key？
**A**: 
- **OpenAI**：访问 https://platform.openai.com/api-keys
- **DeepSeek**：访问 https://platform.deepseek.com
- **其他服务**：查阅相应服务商的文档
